# Don't have a good day, Have a great day 👋

## 안녕하세요, 게임 개발자 조성민입니다.

**라이브 서비스 및 신규 프로젝트 개발 경험이 있는 클라이언트 프로그래머**입니다.  
Unreal Engine C++ 기반으로 게임 시스템을 구현하며,  
**서버 연동, 데이터 기반 구조, UI 성능 최적화**에 관심이 있습니다.

---

## 💼 경험

### 라이브 서비스 게임 개발
- 모바일 라이브 서비스 게임 개발 및 유지보수 참여
- Unreal Engine (C++) 기반 게임플레이 및 UI 시스템 구현
- 클라이언트-서버 통신 처리 및 데이터 연동
- 라이브 환경 버그 대응 및 핫픽스 경험
- UI 및 인벤토리 시스템 성능 최적화

---

### 신규 프로젝트 개발
- 신규 게임 프로젝트 개발 초기 단계 참여
- 게임 시스템 구조 설계 및 기능 구현
- 데이터 기반 시스템 설계 및 확장성 고려
- 프로토타입 제작 및 기능 검증

#### 🔧 인벤토리 UI 성능 최적화
- 가변 슬롯 구조에서 동적 위젯 생성으로 인해 최초 팝업 시 프레임 드랍 발생
- BP 위젯 구조 단순화 및 불필요한 반복 로직 제거 (멱등성 유지)
- 데이터 캐싱을 통한 로직 비용 감소
- Unreal Insights 기반 병목 분석으로 위젯 생성 비용이 주요 원인임을 확인
- 위젯 생성 시점을 분산시켜 초기 프레임 스파이크 완화
